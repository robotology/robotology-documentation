The iCub Launcher.

Description

This application lets the user launch everything is required to work with the iCub in a fully operative framework that embeds the cartesian control, the gaze control as well as the dynamics computed for the whole body.

Instantiated Modules

  • yarprobotinterface
  • iKinCartesianSolver
  • iKinGazeCtrl
  • wholeBodyDynamics
  • gravityCompensator
  • Fingers PID Tuner
  • imuFilter

How to run the Application

Customize the file ./scripts/*.template for your specific platform.
