A canbus sniffer.

Description

Connect to a canbus network and dump all messages to file.

Libraries

  • YARP libraries.
  • A device implementing ICanBus interface (e.g. ecan or pcan).

Parameters

–device device_name: name of the device (e.g. ecan/pcan...)

Tested OS

Linux and Windows.
