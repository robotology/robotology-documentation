A collection of yarpscope instances used to display fingertips skin values.

Description

This applications launches two yarpscope instances, one for the left and one for the right hand. Each yarpscope displays the raw and compensated skin values for all fingertips in the specified hand.

This is a useful application when trying to diagnose issues with the fingertips.

Instantiated Modules

  • yarpscope –context fingertipsPortScope –xml FingertipsPortScopeConf-LeftHandAll.xml
  • yarpscope –context fingertipsPortScope –xml FingertipsPortScopeConf-RightHandAll.xml

How to run the Application

Customize the file ./scripts/*.template for your specific platform.
