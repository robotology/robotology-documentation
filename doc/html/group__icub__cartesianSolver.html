The Cartesian Solvers Launcher.

Description

This application allows the user to launch instances of the iKinCartesianSolver on a specific machine belonging to the cluster. Moreover the yarprobotinterface and the server part of the gaze controller interface are launched as well.

Instantiated Modules

  • yarprobotinterface
  • iKinCartesianSolver
  • iKinGazeCtrl

Configuration Files

The file ./conf/cartesianSolver.ini contains the solvers initialization parameters.

How to run the Application

Customize the file ./scripts/*.template for your specific platform.

  • ./scripts/cartesianSolver.xml.template launches solvers for the following parts: right_arm, left_arm, right_leg, left_leg. as well as the yarprobotinterface and the iKinGazeCtrl.
