◆ calibrate()

bool SpringyFingersModel::calibrate ( const yarp::os::Property &  options)
virtual

Execute the fingers calibration.

Parameters
optionsa Property containing the calibration parameters. Available options are:
finger: a string that encodes the finger to calibrate; it can be "thumb", "index", "middle", "ring", "little", "all", "all_serial", "all_parallel". The special tags "all" and "all_serial" serve to calibrate all the fingers consecutively, whereas the tag "all_parallel" allows calibrating all the fingers at the same time. Moreover, if a list is provided in place of a string - e.g. ("thumb" "little" "middle") - then the specified fingers are calibrated in parallel.
Returns
true/false on success/failure.

Configure the model taking its parameters from a Property object.

Parameters
optionsa Property containing the configuration parameters. Available options are:
name: the name of the model.
type: the handedness type; it can be either "left" or "right".
robot: the name of the robot to connect to; e.g. "icub" or "icubSim".
carrier: the protocol used to connect yarp streaming ports; e.g. "udp", "mcast", "tcp".
verbosity: an integer that accounts for the verbosity level of model print-outs.
