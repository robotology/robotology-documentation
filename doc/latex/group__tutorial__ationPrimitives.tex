An example module that makes use of \mbox{\hyperlink{group__ActionPrimitives}{action\+Primitives}} library in order to execute a sequence of simple actions\+: reaches for an object, tries to grasp it, then lifts it and finally releases it.

1) A bottle containing the 3-\/d position of the object to grasp is received (let be x1).

2) To this 3-\/d point a systematic offset is added in order to compensate for the uncalibrated kinematics of the arm (let be x2=x1+systematic\+\_\+offset).

3) The robot reaches for a position located on top of the object with the specified orientation (let be reach(x2+grasp\+\_\+disp,o)).

4) The robot grasps the object (let be reach(x2,o)).

5) The hand is closed.

6) The robot lifts the object to a specified location (let be reach(x2+lift\+\_\+displacement,o)).

7) The robot releases the grasped object.

8) The robot steers the arm to home position.
