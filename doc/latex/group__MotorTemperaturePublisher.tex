\hypertarget{group__tutorial__perceptiveModels_intro_sec}{}\doxysubsection{Description}\label{group__tutorial__perceptiveModels_intro_sec}
This module is a working example for building a YARP device that exploits the i\+Motor YARP interfaces to interact with the motion control module. Specifically to this example, we are defining a {\itshape {\bfseries{Poly\+Driver}}} device that interacts with the pointer to the {\itshape {\bfseries{i\+Motor}}} yarp interface and prints the motor temperatures to an output port. Those temperatures values can be obtained exploiting the interface method {\itshape get\+Temperatures()} and saved to a c-\/array of doubles.

In order to use this module, whose final target is to print at the port called {\itshape {\bfseries{/portprefix/motor\+\_\+temperatures\+:o}}} the temperatures of the motors connected, the following steps should be done\+: 
\begin{DoxyItemize}
\item supposed to have the correct setup running on the yarpserver sending data to state ports  
\item \mbox{\hyperlink{classMotorTemperaturePublisher}{Motor\+Temperature\+Publisher}} --from config.\+ini  
\end{DoxyItemize}

This will instantiate the aformentioned port where the following values will be streamed at the specified period\+:


\begin{DoxyItemize}
\item time\+Stamp $\vert$ temperature\+\_\+motor\+\_\+0 $\vert$ boolean\+\_\+limit\+\_\+overcame\+\_\+0 $\vert$ ... $\vert$ temperature\+\_\+motor\+\_\+N $\vert$ boolean\+\_\+limit\+\_\+overcame\+\_\+N $\vert$  
\end{DoxyItemize}\hypertarget{group__tutorial__perceptiveModels_parameters_sec}{}\doxysubsection{Parameters}\label{group__tutorial__perceptiveModels_parameters_sec}

\begin{DoxyItemize}
\item --context\+: select the current context.
\item --from\+: configuration file name.
\item --name\+: module\textquotesingle{}s name; all the open ports will be tagged with the prefix /name.
\item --robotname\+: name of the robot to connect to.
\item --portprefix\+: name of the part\textquotesingle{}s port to connect to.
\item --period\+: period used for streaming the data to the port.
\item --listofjoints\+: joints, as list a list, i.\+e. (0 1 2), for which you want to get the temperature.
