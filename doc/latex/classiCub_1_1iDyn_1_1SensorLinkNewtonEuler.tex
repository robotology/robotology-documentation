\doxysubsection{Detailed Description}
A class for setting a virtual sensor link. 

This class is used to initialize the forward and backward phase of Newton-\/\+Euler\textquotesingle{}s method in the Inverse formulation, but also to have estimation of the FT measures by the sensor; the sensor frame is defined with respect to the i-\/th link, where the sensor is attached. Note that this is a virtual link, since there\textquotesingle{}s no \mbox{\hyperlink{classiCub_1_1iDyn_1_1iDynLink}{i\+Dyn\+Link}} attached\+: it is just necessary to make the recursive Inverse Newton-\/\+Euler computations; however, inertia, mass and COM are defined, for the portion of link defined between sensor and i-\/th frame.
