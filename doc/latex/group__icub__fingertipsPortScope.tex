A collection of yarpscope instances used to display fingertips skin values. 

\hypertarget{group__tutorial__perceptiveModels_intro_sec}{}\doxysubsection{Description}\label{group__tutorial__perceptiveModels_intro_sec}
This applications launches two yarpscope instances, one for the left and one for the right hand. Each yarpscope displays the raw and compensated skin values for all fingertips in the specified hand.

This is a useful application when trying to diagnose issues with the fingertips.\hypertarget{group__icub__iCubStartup_int_sec}{}\doxysubsection{Instantiated Modules}\label{group__icub__iCubStartup_int_sec}
