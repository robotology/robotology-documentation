\doxysubsection{Detailed Description}
\mbox{\hyperlink{classTools_1_1Emb__RensponseTimingVerifier}{Tools\+::\+Emb\+\_\+\+Rensponse\+Timing\+Verifier}}. 

This class let you to analyze the needed time of a request to get a response during a given amount of time. This class has been developed with the idea in mind of monitoring the the response time of a request along a period in order to understand how many situations there are jitter, other than to calculate the medium value. This class needs to be initialize and then you need to add the current response time and its relative absolute request time. The class prints the histogram of real response timing every @report\+Period, by yarp logger mechanism and resets the data collected until now.
