Display a kinematic model of the robot from the encoders. It also shows a representation of the output of the inertial sensor.

\hypertarget{group__icub__gui_visobj_sec}{}\doxysubsection{Vision objects}\label{group__icub__gui_visobj_sec}
i\+Cub\+Gui can also display data blobs from vision modules, representing them as 3D ellipsoids characterized by name, dimensions, color, transparency, position and orientation. Objects in i\+Cub\+Gui are managed by a port specified in the i\+Cub\+Gui.\+ini configuration file with the tag {\itshape objport}, as follows\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{robot /icubSim}
\DoxyCodeLine{geometry skeleton.ini}
\DoxyCodeLine{objport /iCubGui/objects}

\end{DoxyCode}


To add an object to the display object list, send to i\+Cub\+Gui a bottle composed as following on the port specified in i\+Cub\+Gui.\+ini\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{yarp::os::Bottle obj;}
\DoxyCodeLine{}
\DoxyCodeLine{obj.addString(\textcolor{stringliteral}{"{}object"{}}); \textcolor{comment}{// command to add/update an object}}
\DoxyCodeLine{obj.addString(\textcolor{stringliteral}{"{}my\_object\_name"{}});}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{comment}{// object dimensions in millimiters}}
\DoxyCodeLine{\textcolor{comment}{// (it will be displayed as an ellipsoid with the tag "{}my\_object\_name"{})}}
\DoxyCodeLine{obj.addFloat64(dimX);}
\DoxyCodeLine{obj.addFloat64(dimY);}
\DoxyCodeLine{obj.addFloat64(dimZ);}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{comment}{// object position in millimiters}}
\DoxyCodeLine{\textcolor{comment}{// reference frame: X=fwd, Y=left, Z=up}}
\DoxyCodeLine{obj.addFloat64(posX);}
\DoxyCodeLine{obj.addFloat64(posY);}
\DoxyCodeLine{obj.addFloat64(posZ);}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{comment}{// object orientation (roll, pitch, yaw) in degrees}}
\DoxyCodeLine{obj.addFloat64(rotX);}
\DoxyCodeLine{obj.addFloat64(rotY);}
\DoxyCodeLine{obj.addFloat64(rotZ);}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{comment}{// object color (0-\/255)}}
\DoxyCodeLine{obj.addInt32(R);}
\DoxyCodeLine{obj.addInt32(G);}
\DoxyCodeLine{obj.addInt32(B);}
\DoxyCodeLine{\textcolor{comment}{// transparency (0.0=invisible 1.0=solid)}}
\DoxyCodeLine{obj.addFloat64(alpha);}

\end{DoxyCode}


To delete an object\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{yarp::os::Bottle obj;}
\DoxyCodeLine{}
\DoxyCodeLine{obj.addString(\textcolor{stringliteral}{"{}delete"{}});}
\DoxyCodeLine{obj.addString(\textcolor{stringliteral}{"{}my\_object\_name"{}});}

\end{DoxyCode}


To reset the object list\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{yarp::os::Bottle obj;}
\DoxyCodeLine{}
\DoxyCodeLine{obj.addString(\textcolor{stringliteral}{"{}reset"{}});}

\end{DoxyCode}
