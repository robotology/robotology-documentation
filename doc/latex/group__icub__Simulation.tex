\item Has been designed to reproduce, as accurately as possible, the physics and the dynamics of the robot and its environment
\item It has been constructed collecting data directly from the robot design specifications in order to achieve an exact replication
\item The software infrastructure and inter-\/process communication are as those used to control the physical robot
