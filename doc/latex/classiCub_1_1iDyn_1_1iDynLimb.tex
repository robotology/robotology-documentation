\doxysubsection{Detailed Description}
A class for defining a generic Limb within the \mbox{\hyperlink{namespaceiCub_1_1iDyn}{i\+Dyn}} framework. 

Note\+: Since i\+Kin\+Limb is derived with protected modifier from i\+Kin\+Chain in order to make some methods hidden to the user such as add\+Link, rm\+Link and so on, all the remaining public methods have been redeclared and simply inherited. \mbox{\hyperlink{classiCub_1_1iDyn_1_1iDynLimb}{i\+Dyn\+Limb}} follows the same rule of i\+Kin\+Limb, therefore the methods must be redeclared.
