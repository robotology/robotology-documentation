Perform outliers detection over the provided data. 

~\newline
 Only one element is considered at a time, unless the {\itshape recursive} option is specified. 
\begin{DoxyParams}{Parameters}
{\em data} & contains points to be verified. \\
\hline
{\em options} & contains detection options. If the properties {\itshape mean} and {\itshape std} are provided with corresponding doubles, then the computation of relative values is skipped. If the property {\itshape sorted} is provided, then the data are expected to be sorted either in ascending or descending order. If the property {\itshape recursive} is specified, then the detection is carried out over and over on the outcome of the previous instance until no more outliers are found. \\
