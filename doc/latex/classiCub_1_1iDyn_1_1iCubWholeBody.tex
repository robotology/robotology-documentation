\doxysubsection{Detailed Description}
A class for connecting Upper\+Torso and Lower\+Torso of the \mbox{\hyperlink{namespaceiCub}{i\+Cub}}, then getting the Whole\+Body in the dynamic framework. 

It is merely a container\+: pointers to upper and lower torso objects are accessible, so all public methods of the two objects can be used.
