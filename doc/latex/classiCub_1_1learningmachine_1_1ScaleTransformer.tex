\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classiCub_1_1learningmachine_1_1ScaleTransformer}{Scale\+Transformer}} is a \mbox{\hyperlink{classiCub_1_1learningmachine_1_1ITransformer}{ITransformer}} that supports element-\/based scaling transformations. 

These transformations include standardization, normalization and other kind of linear scalings based on an offset and scale. The type and configuration can be set for individual elements in the input vectors.
