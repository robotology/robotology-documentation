\doxysubsection{Detailed Description}
A derived class defining a first abstraction layer on top of \mbox{\hyperlink{group__ActionPrimitives}{action\+Primitives}} father class. 

It internally predeclares (without actually defining) a set of hand sequence motions key (\char`\"{}open\+\_\+hand\char`\"{}, \char`\"{}close\+\_\+hand\char`\"{} and \char`\"{}karate\+\_\+hand\char`\"{} \+:) that are used for \mbox{\hyperlink{classiCub_1_1action_1_1ActionPrimitivesLayer1_a95ab2896f4f80d8ef01f12ffc1b8aa74}{grasp()}}, \mbox{\hyperlink{classiCub_1_1action_1_1ActionPrimitivesLayer1_afaffe75f376d9b03053eee1ee62902ed}{touch()}} and \mbox{\hyperlink{classiCub_1_1action_1_1ActionPrimitivesLayer1_a94b4da629121987844332c75d4d52300}{tap()}} actions.

\begin{DoxyNote}{Note}
Given as an example of how primitive actions can be combined in higher level actions, thus how further layers can be inherited from the base class. 
\end{DoxyNote}
