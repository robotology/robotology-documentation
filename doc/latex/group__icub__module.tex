\doxysubsection{Detailed Description}
This is the list of \mbox{\hyperlink{namespaceiCub}{i\+Cub}} modules, i.\+e. executables in the icub-\/main/src directory. To add your own module to this list, just add a comment like the following anywhere in your source code (replacing \char`\"{}your\+\_\+module\+\_\+name\char`\"{} with the name of your module)\+: \begin{DoxyVerb}  /**
   *
   * \defgroup icub_your_module_name your_module_name
   * @ingroup icub_module
   *
   */
\end{DoxyVerb}


For convenience, try to keep the module name the same as the directory it is placed in. If you\textquotesingle{}ve a good reason to do something different, of course you can do that too.
