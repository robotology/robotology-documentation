\doxysubsection{Detailed Description}
Reply processor helper class for predictions. 

This processor receives requests for predictions, transforms the samples and relays the request to the associated port. This architecture allows multiple transformers to be daisy chained, as long as the chain is terminated by a \mbox{\hyperlink{classiCub_1_1learningmachine_1_1PredictModule}{Predict\+Module}}.
