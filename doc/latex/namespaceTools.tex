\doxysubsection{Detailed Description}
In the \mbox{\hyperlink{namespaceTools}{Tools}} namespace there are classes useful to check some kinds of performance on robot. 

Currently, the available classes are \mbox{\hyperlink{classTools_1_1Emb__PeriodicEventVerifier}{Emb\+\_\+\+Periodic\+Event\+Verifier}} and \mbox{\hyperlink{classTools_1_1Emb__RensponseTimingVerifier}{Emb\+\_\+\+Rensponse\+Timing\+Verifier}}\+: the first let you to verify the frequency of a periodic event, while the second let you to analyze the trend of response time to a request. This class are based on classes belonging to em\+BEDDED RObot library of C++11, so these classes are very suitable for working with high precision time, like 1 millisecond.
