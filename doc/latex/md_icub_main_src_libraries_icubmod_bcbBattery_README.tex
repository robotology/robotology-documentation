This module implements the \href{http://www.yarp.it/git-master/classyarp_1_1dev_1_1IBattery.html}{\texttt{ {\ttfamily IBattery}}} interface to retrieve the status of the battery in the \mbox{\hyperlink{namespaceiCub}{i\+Cub}} backpack.\hypertarget{md_icub_main_src_libraries_icubmod_bcbBattery_README_autotoc_md1}{}\doxysection{Dependencies}\label{md_icub_main_src_libraries_icubmod_bcbBattery_README_autotoc_md1}
In order for this module to be compiled, it needs to be enabled with the option {\ttfamily ENABLE\+\_\+icubmod\+\_\+bcb\+Battery} set to {\ttfamily ON}. In addition, {\ttfamily yarp} needs to be compiled with the following options set to {\ttfamily ON}\+:


\begin{DoxyItemize}
\item {\ttfamily ENABLE\+\_\+yarpmod\+\_\+battery\+Wrapper}
\item {\ttfamily ENABLE\+\_\+yarpmod\+\_\+serial}
\item {\ttfamily ENABLE\+\_\+yarpmod\+\_\+serialport}
\end{DoxyItemize}

In addition, in order to have a GUI representation of the battery status, enable the following in {\ttfamily yarp}


\begin{DoxyItemize}
\item {\ttfamily ENABLE\+\_\+yarpmod\+\_\+battery\+Client}
\item {\ttfamily YARP\+\_\+\+COMPILE\+\_\+yarpbatterygui}
\end{DoxyItemize}\hypertarget{md_icub_main_src_libraries_icubmod_bcbBattery_README_autotoc_md2}{}\doxysection{How to use}\label{md_icub_main_src_libraries_icubmod_bcbBattery_README_autotoc_md2}
This module can be used by resorting to the configuration files already available in the \href{https://github.com/robotology/robots-configuration/tree/master/experimentalSetups/battery}{\texttt{ {\ttfamily battery} folder}} in the {\ttfamily experimental\+Setups} of {\ttfamily robots-\/configuration}. From that folder, it is sufficient to run {\ttfamily yarprobotinterface}.\hypertarget{md_icub_main_src_libraries_icubmod_bcbBattery_README_autotoc_md3}{}\doxysection{Configuration parameters}\label{md_icub_main_src_libraries_icubmod_bcbBattery_README_autotoc_md3}
The configuration file of this module needs two groups. A group named {\ttfamily SERIAL\+\_\+\+PORT} with all the parameters relative to the serial port connection. For the meaning of these parameters refer to the {\ttfamily serial\+Port} \href{https://github.com/robotology/yarp/blob/ab488c74e37de629af207b83be2fd90ae7412b1b/src/devices/serialport/SerialDeviceDriver.h\#L34-L70}{\texttt{ header file}}. Their setup depends on the serial configuration done on the hardware side.

The second group should be named {\ttfamily GENERAL} and should contain the following\+:


\begin{DoxyItemize}
\item {\ttfamily thread\+\_\+period}\+: the thread period in milliseconds
\item {\ttfamily screen}\+: when equal to 1, it enables an info print in the terminal showing the battery status
\item {\ttfamily verbose}\+: when equal to 1, it prints on the terminal the raw value coming from the terminal.
\item {\ttfamily silence\+\_\+sync\+\_\+warnings}\+: when 1 it avoids printing warning messages relative to syncing. 
\end{DoxyItemize}
