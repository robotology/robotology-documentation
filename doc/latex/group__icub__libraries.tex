\doxysubsection{Detailed Description}
This is a list of static libraries. 

Usually in icub-\/main/src, they are collected here in a group to separate them from the executables (in \mbox{\hyperlink{group__icub__module}{i\+Cub modules}} \textbackslash{}endref).

To add a new library to this group add this to your code\+:

\begin{DoxyVerb}  /**
   * \defgroup icub_your_library your_library
   * @ingroup icub_libraries
   *
   */
\end{DoxyVerb}
