\doxysubsection{Detailed Description}
A class that represents a prediction result. 

The serialization of this class is compatible with Portable\+Pair$<$\+Vector,\+Vector$>$, such that standard Yarp classes can be used to read a \mbox{\hyperlink{classiCub_1_1learningmachine_1_1Prediction}{Prediction}} from the network.
