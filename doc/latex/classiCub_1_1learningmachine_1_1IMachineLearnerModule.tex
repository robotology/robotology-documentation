\doxysubsection{Detailed Description}
An abstract base module for the machine learning YARP interface. 

This abstract class contains base functionality that is shared by \mbox{\hyperlink{classiCub_1_1learningmachine_1_1PredictModule}{Predict\+Module}}, \mbox{\hyperlink{classiCub_1_1learningmachine_1_1TrainModule}{Train\+Module}} and \mbox{\hyperlink{classiCub_1_1learningmachine_1_1TransformModule}{Transform\+Module}}.
