Get the raw buffer from the frame grabber. 

The driver returns a copy of the internal memory buffer acquired by the frame grabber, no post processing is applied (e.\+g. no color reconstruction/demosaicking). The user must allocate the buffer; the size of the buffer, in bytes, is determined by calling \mbox{\hyperlink{classyarp_1_1dev_1_1StaticGrabber_a62cbd77fcb7e2fec1a0128fc202a5c8b}{get\+Raw\+Buffer\+Size()}}.
