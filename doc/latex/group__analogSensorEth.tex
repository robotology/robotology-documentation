Driver for Ethernet communication with Force-\/\+Torque sensors, the Strain2 board. 

Driver for Ethernet communication with inertial sensors (accelerometers, gyroscopes) mounted on Strain2 board.

It is also possible read the temperature of tthe sensor.

The parameters accepted in the config argument of the open method are described in \href{http://wiki.icub.org/wiki/Robot_configuration}{\texttt{ http\+://wiki.\+icub.\+org/wiki/\+Robot\+\_\+configuration}}.

It is possible to read from multiple strain2 board. The parameters accepted in the config argument of the open method are described in \href{http://wiki.icub.org/wiki/Robot_configuration}{\texttt{ http\+://wiki.\+icub.\+org/wiki/\+Robot\+\_\+configuration}}.
