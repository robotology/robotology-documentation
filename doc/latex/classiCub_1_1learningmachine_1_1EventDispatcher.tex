\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classiCub_1_1learningmachine_1_1EventDispatcher}{Event\+Dispatcher}} manages the relation between the various instances of IEvent\+Listeners and IEvents. 

It allows IEvents to get raised and dispatches these to the registered IEvent\+Listeners. Internally, the \mbox{\hyperlink{classiCub_1_1learningmachine_1_1EventDispatcher}{Event\+Dispatcher}} uses a double dispatching mechanism that allows extension of both IEvent\+Listeners and IEvents.
