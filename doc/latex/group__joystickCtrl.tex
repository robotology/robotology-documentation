The module creates the port /joystick\+Ctrl\+:o used to transmit the joystick data. The output of the port consists in a sequence of $<$n$>$ doubles ($<$n$>$ depending on the number of axes specified in the configuration file) containing the readings. The port /joystick\+Ctrl/axis\+:o contains the axis output only (raw data).
