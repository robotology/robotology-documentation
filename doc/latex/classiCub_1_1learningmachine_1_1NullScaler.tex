\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classiCub_1_1learningmachine_1_1NullScaler}{Null\+Scaler}} is a scaler that does nothing, the output of the transform function is equal to its input. 

The use is so that the \mbox{\hyperlink{classiCub_1_1learningmachine_1_1ScaleTransformer}{Scale\+Transformer}} can \textquotesingle{}disable\textquotesingle{} transformation on certain inputs.
