\doxysubsection{Detailed Description}
An generalized interface for a learning machine with a fixed domain and codomain size. 

Many machine learning techniques will fall in this category.

The learning machine can be used for regression and classification from R$^\wedge$M to R$^\wedge$N. M and N are here fixed sizes which should be respected by the samples that are fed into the system.
