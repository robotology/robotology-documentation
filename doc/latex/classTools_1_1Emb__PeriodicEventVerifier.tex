\doxysubsection{Detailed Description}
\mbox{\hyperlink{classTools_1_1Emb__PeriodicEventVerifier}{Tools\+::\+Emb\+\_\+\+Periodic\+Event\+Verifier}}. 

This class let you to verify if a periodic event is triggered with the desired frequency. After its initialization, you need to tick it every event occurrence. The class prints the histogram of effective event timing every @report\+Period, by yarp logger mechanism and resets the data collected until now.
